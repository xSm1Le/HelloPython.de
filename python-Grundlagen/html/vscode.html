▸ 01 — Download

VS Code herunterladen

Gehe auf code.visualstudio.com und klicke auf den Download-Button. Die Seite erkennt dein Betriebssystem automatisch.

🪟
Windows
.exe Installer
🍎
macOS
.dmg Datei
🐧
Linux
.deb / .rpm
💡 Tipp: Wähle immer die stabile Version (Stable Build) – nicht die Insiders-Version.
▸ 02 — Installation

Installer ausführen

Führe die heruntergeladene Datei aus. Diese Optionen solltest du beim Setup unbedingt aktivieren:

✓
"In PATH hinzufügen" aktivieren
Damit kannst du code . im Terminal nutzen
✓
"Zu Kontextmenü hinzufügen" aktivieren
Rechtsklick auf Ordner → "Mit Code öffnen"
✓
Desktop-Verknüpfung erstellen
Schnellzugriff direkt vom Desktop
⚠️ Wichtig: Nach der Installation den PC einmal neu starten, damit alle PATH-Änderungen wirksam werden.

▸ 04 — Interpreter

Python-Interpreter auswählen

Nach der Extension-Installation musst du VS Code mitteilen, welche Python-Version es nutzen soll.

①
Drücke Ctrl + Shift + P um die Command Palette zu öffnen
②
Tippe Python: Select Interpreter und drücke Enter
③
Wähle die installierte Python-Version aus der Liste – z.B. Python 3.12.x
💡 Die ausgewählte Python-Version siehst du danach unten links in der Statusleiste von VS Code.
▸ 05 — Ausführen

Erstes Programm ausführen

Erstelle eine neue Datei und führe dein erstes Python-Skript direkt in VS Code aus.

①
Erstelle eine neue Datei: Ctrl + N → Speichern als hallo.py
②
Tippe folgenden Code ein:
hallo.py
print("Hallo, VS Code!") name = input("Wie heißt du? ") print(f"Willkommen, {name}!")
③
Klicke oben rechts auf ▶ Run oder drücke F5
TERMINAL
$ python hallo.py
Hallo, VS Code!
Wie heißt du? █

Wichtige Shortcuts

Ctrl+Shift+P

Ctrl+/

Alt+↑↓
Zeile verschieben
